== Use Case 

The following diagram

....

│                   ┌─────────────────────┐
├─ not_synched    ◄─┤ This folder is      │
│                   │ excluded            │
│                   └─────────────────────┘
│
│                   ┌─────────────────────────────────┐
└─ synched        ◄─┤ This folder and the sub folders │
    │               │  will be synched to AWS S3      │
    │               └─────────────────────────────────┘

    ├─ backup2

....
